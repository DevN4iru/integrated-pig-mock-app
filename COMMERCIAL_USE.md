# Pigstep Commercial Use

Pigstep may be used commercially only by the project owner or by parties who have written permission from the project owner.

Allowed commercial models may include:

- private farm deployment;
- client-managed deployment;
- hosted SaaS subscription;
- paid setup and support;
- licensing under a separate written agreement;
- custom development under a separate written agreement.

Receiving access to the source code does not grant commercial resale, sublicensing, white-labeling, public distribution, or competing-product rights.

Customer access to a hosted Pigstep instance does not grant access to the source code or ownership of the software.
